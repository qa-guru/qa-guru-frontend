## Running frontend

### In development mode

```shell
yarn run dev
```

### Build project

```shell
yarn run build
```

### Preview project

```shell
yarn run preview
```

## Running graphql-codegen

```shell
yarn run gen
```

## Running storybook

### In development mode

```shell
yarn run storybook
```

### Build storybook

```shell
yarn run build-storybook
```

## Running husky

```shell
yarn run husky
```
